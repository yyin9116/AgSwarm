# AgSwarm Client v0.2.19

This release stabilizes the packaged desktop runtime for the Ag assistant.

## Fixed

- Bundles pi/pi-web from productized npm dependencies instead of requiring a sibling pi checkout during release builds.
- Fixes Windows NSIS packaging for the bundled pi runtime by building from a short mapped path.
- Preserves the bundled pi-web diagnostic package while pruning runtime-only development files.

## Verification

- Desktop CI passed on GitHub Actions.
- Manual release workflow passed for macOS Apple Silicon and Windows x64.
