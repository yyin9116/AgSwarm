# AgSwarm Client v0.2.21

This release repairs the desktop release workflow after the v0.2.20 tag was blocked in CI.

## Fixed

- Keeps the Linux-only Rust check from preparing the full pi-web runtime, which is only packaged for supported desktop release targets.
- Preserves full pi-web runtime preparation and validation for macOS Apple Silicon and Windows x64 release builds.
- Carries forward the v0.2.20 Windows-ready runtime, native chat rendering, file/device transfer, tool-call folding, and theme/profile improvements.

## Verification

- `npm run lint`
- `npm run build`
- `npm run stress:pi-web`
- `TAURI_SIDECAR_TARGET=aarch64-apple-darwin cargo check`
