# AgSwarm v0.2.14

## Auto-Update

- Adds the signed Tauri updater path for the desktop app.
- Settings now includes a Software Updates card for checking, downloading, installing, and relaunching into a new release.
- Release builds now require `TAURI_SIGNING_PRIVATE_KEY` and publish updater `.sig` files plus `latest.json`.

## Release Packaging

- Keeps the public release matrix focused on macOS Apple Silicon and Windows x64.
- Keeps release artifact names short and product-facing.
- Fails release packaging if expected signed updater artifacts are missing.

## Verification

- Desktop CI passed on GitHub for the implementation commit.
- Local TypeScript check passed.
- Updater manifest generation and artifact collection were smoke tested locally.
