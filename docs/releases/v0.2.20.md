# AgSwarm Client v0.2.20

This release focuses on Windows testing readiness and cross-platform AgSwarm desktop communication.

## Added

- Adds the native pi-web capability panel to the chat page for model, thinking, command, file, git, terminal, and auth operations.
- Adds file attachment and device file-drop flows so files can be staged from chat or sent to a discovered device.
- Adds persistent profile/theme settings, including follow-system appearance mode.

## Fixed

- Packages the pi-web runtime, client assets, plugins, and Node sidecar for Windows without requiring a separate pi-web install.
- Keeps the chat composer visible by bounding the native chat layout and tool panel.
- Uses pi-web-style Markdown rendering for chat bubbles, including links, lists, inline code, fenced code blocks, and copyable code blocks.
- Collapses repeated tool calls into a compact status row and limits detail expansion to one tool at a time.
- Reduces device polling outside the Devices page to avoid tab-switching stalls.
