# AgSwarm v0.2.15

## Auto-Update

- Fixes release packaging for Tauri signed auto-update artifacts.
- Publishes the macOS Apple Silicon updater bundle as `.app.tar.gz` with its generated `.sig` file.
- Keeps the macOS DMG as the manual installer while `latest.json` points at the signed updater bundle.

## Verification

- Desktop CI and platform packaging are expected to run from the `v0.2.15` tag.
- The updater manifest generator was locally smoke-tested against the GitHub Actions artifact layout.
