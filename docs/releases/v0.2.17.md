# AgSwarm v0.2.17

## Windows Packaging

- Prepares the bundled pi-web runtime during Tauri packaging so Windows builds do not depend on a host-installed `@jmfederico/pi-web` package.
- Validates pi-web server, session daemon, client assets, plugins, and Windows `node-pty` native files before release artifacts are built.
- Suppresses the extra Windows console window for release builds and managed background processes.

## Verification

- Release validation must fail if packaged pi-web runtime resources are missing.
- Windows validation should install the new setup executable and confirm Chat loads without `PI_WEB_PACKAGE_DIR` or an external Node/pi-web install.
