# AgSwarm v0.2.8

## Fixes
- Fixed Windows CI build failure caused by shell incompatibility in `Prepare icon compatibility path` step.
- Switched that step to `shell: python` so it runs consistently on both Windows and macOS runners.

## Packaging
- macOS (Apple Silicon): `AgSwarm-macos-arm64.dmg`
- Windows: `AgSwarm-windows-x64.zip`
- Checksums: `SHA256SUMS.txt`

## Notes
- This release focuses on restoring stable cross-platform release pipeline execution.
