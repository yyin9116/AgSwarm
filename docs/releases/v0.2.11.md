# AgSwarm v0.2.11

## Chat
- Added current app state context so Ag can answer questions about visible devices without inspecting source files.
- Collapsed consecutive tool events into compact status rows by default.
- Prevented internal runtime context, tool execution parts, and streaming duplicates from leaking into the final chat bubble.
- Tuned thinking and tool-call typography to match the lighter Command page style.

## Packaging
- Bumped desktop metadata to `0.2.11` for package, Tauri, and Cargo release outputs.
- Removed the local development workspace path from the default chat workspace.

## Distribution
- GitHub Actions builds macOS Apple Silicon, macOS Intel, Windows x64, and Linux x64 release artifacts from the `v0.2.11` tag.
