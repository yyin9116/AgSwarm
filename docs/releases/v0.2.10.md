# AgSwarm v0.2.10

## Fixes
- Desktop Settings tab layout reworked with scrollable form and better spacing, resolving crowded/stacked display on macOS.
- Connect/Refresh feedback improved:
  - Immediate `Connecting...` / `Refreshing...` status.
  - Explicit success/failure state in header and connection status panel.
  - Clear diagnostic message with target host:port and troubleshooting hint when NATS connection fails.
- Refresh logic now validates NATS connection first and no longer silently swallows all transport errors as generic offline nodes.
- Reconnect during runtime settings apply now surfaces connection errors instead of failing silently.

## i18n
- Replaced corrupted zh-CN translation map with valid Simplified Chinese text.
- Added and corrected common Settings/Connection labels.

## Packaging
- macOS (Apple Silicon): `AgSwarm-macos-arm64.dmg`
- Windows: `AgSwarm-windows-x64.zip`
- Checksums: `SHA256SUMS.txt`
