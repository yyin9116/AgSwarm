# AgSwarm v0.2.9

## Fixes
- Fixed GitHub Release publish failure in `Prepare release notes body`.
- Switched release-note injection to `body_path` mode, avoiding fragile multiline `$GITHUB_OUTPUT` handling.
- Keep fallback logic: if `docs/releases/<tag>.md` is missing, workflow still uses auto-generated release notes.

## Packaging
- macOS (Apple Silicon): `AgSwarm-macos-arm64.dmg`
- Windows: `AgSwarm-windows-x64.zip`
- Checksums: `SHA256SUMS.txt`
