# AgSwarm v0.2.6

## Highlights
- Fixed desktop startup config precedence issue: when `settings.json` contains a custom `nats_url`, runtime now reconnects using that value instead of staying on default loopback (`127.0.0.1`).
- Improved NATS connection diagnostics:
  - URL normalization/validation (scheme/host/port checks).
  - Faster fail behavior on unavailable servers.
  - Password masking in transport logs.
- Improved connection UX in desktop:
  - Clearer connect/refresh feedback state in UI.
  - Better startup connect failure notification.
- macOS UI fix: increased `QLineEdit`/`QComboBox` minimum height and padding to avoid text clipping in Settings.

## Packaging
- macOS (Apple Silicon): `AgSwarm-macos-arm64.dmg`
- Windows: `AgSwarm-windows-x64.zip`
- Checksums: `SHA256SUMS.txt`

## Notes
- This release focuses on connection stability and observability for Mac/Win LAN linking tests.
- Existing task/adapters flow remains unchanged.
